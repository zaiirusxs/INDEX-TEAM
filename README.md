🌿 WellnessPro – Meet Our Team Webpage
📖 Short Description

WellnessPro is a static wellness-themed website that introduces a professional support team dedicated to personal growth and self-improvement. The webpage features a clean layout, structured navigation bar, hero section, and a responsive team member showcase.

The design focuses on clarity, visual organization, and a warm color palette to reflect encouragement, support, and empowerment.

✨ Features / Functionality

🏠 Header with brand logo

📑 Navigation bar (Home, About, The Team, Blog, Contact Us)

🎯 Hero section with motivational message

👥 Team member showcase with photos and roles

🎨 Styled layout using internal CSS

📱 Responsive flexbox team container

📄 Footer section

🛠️ Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout design

Flexbox – Responsive team layout

🚀 Installation / Usage Instructions
Requirements

Any modern web browser (Chrome, Edge, Firefox, etc.)

Steps to Run

Save the file as index.html.

Place the required image files in the same project folder.

Double-click the file
OR
Open it in your preferred web browser.

No additional installation is required.

🎯 Purpose

This project was developed to demonstrate:

Website structure using semantic HTML

Layout styling using CSS

Flexbox implementation for responsive design

UI organization and color coordination

Content presentation for team-based organizations

It showcases foundational front-end web development skills.
